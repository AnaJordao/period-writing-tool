# ✏️ Period - Writing Tool

Period is an application that helps to manage writing sessions easily.

## How to run?

### Frontend

First install the dependencies:

```bash

npm i

```

Now you can continue to:

#### Run

```bash

npm run dev

```
#### Test

```bash

npm run test

```

### Backend

First, enter the `backend` folder and install the dependencies:

```bash

cd ./backend
npm i

```

Now you can continue to:

#### Run

```bash

npm run start:dev

```

#### Test

```bash

npm run test:watch

```
